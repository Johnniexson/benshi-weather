# benshi-weather

## Project setup

```
npm install
```

Create `.env.local` file and add openweathermap apiKey with property name of `VUE_APP_WEATHER_API_KEY`

### Compiles and hot-reloads for development

```
npm run serve
```

### Compiles and minifies for production
